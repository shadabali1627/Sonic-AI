# Mobile App - Running Instructions

To run this Flutter application, you can use the following commands from the root of this directory (`d:\sonic ai\mobile app`):

## Run on Devices

### listing Available Devices
Check which devices are connected or available:
```bash
flutter devices
```

### Run on Windows
To run the app as a Windows desktop application:
```bash
flutter run -d windows
```

### Run on Web (Chrome)
To run the app on the web:
```bash
flutter run -d chrome
```

### Run on Android Emulator or Device
Ensure an Android emulator is running or a device is connected via USB with developer mode enabled.
```bash
flutter run
```
(It will auto-select if only one device is connected, or prompt you to choose).

## Development
- **Hot Reload**: Press `r` in the terminal while the app is running to reload changes instantly.
- **Hot Restart**: Press `R` to restart the app completely.
